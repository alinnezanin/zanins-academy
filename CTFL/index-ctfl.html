Curso CTFL 4.0

Certified Tester Foundation Level

Bem-vindo(a) ao curso!

Este curso foi desenvolvido com base no syllabus oficial da certificação ISTQB CTFL 4.0. Cada aula aborda um capítulo completo do syllabus, com material de apoio, roteiro de vídeo, quiz de fixação e identidade visual padronizada para facilitar o aprendizado.

Objetivo do Curso

Preparar os alunos para a certificação CTFL 4.0, promovendo uma compreensão sólida dos fundamentos de testes de software e suas boas práticas, com conteúdo acessível e aplicável ao dia a dia profissional.

Metodologia

  • 🎬 Uma videoaula por capítulo
  • 🧾 Roteiro detalhado de apresentação
  • 📝 Quiz de fixação por módulo
  • 🎨 Identidade visual única para todos os materiais
  • 🌐 Deploy via GitHub Pages
